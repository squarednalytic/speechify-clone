Text to Speech Reader - Ekstensi Chrome

Ekstensi Chrome yang mengubah teks menjadi suara, mirip dengan Speechify.

Cara Install
Metode 1: Install Manual (Developer Mode)

# 1. Download atau clone repository
git clone https://github.com/yourusername/speechify-clone.git

# 2. Buka Chrome dan ketik:
chrome://extensions/

# 3. Aktifkan "Developer mode" (pojok kanan atas)

# 4. Klik "Load unpacked" dan pilih folder hasil download

Metode 2: File ZIP

1. Download semua file
2. Buat folder bernama "tts-reader"
3. Masukkan semua file ke folder tersebut
4. Kompress folder menjadi ZIP
5. Drag & drop file ZIP ke chrome://extensions/

Cara Penggunaan
Cara 1: Melalui Klik Kanan
1. Buka website manapun
2. Blok teks yang ingin dibaca
3. Klik kanan ‚Üí "Baca teks yang dipilih"
   Atau klik kanan di halaman ‚Üí "Baca seluruh halaman"

Cara 2: Melalui Icon Extension
1. Klik icon extension di toolbar Chrome
2. Pilih teks di website atau masukkan teks sendiri
3. Klik tombol Play

Cara 3: Keyboard Shortcut
Windows/Linux: Ctrl+Shift+P (Play/Pause)
                Ctrl+Shift+S (Stop)

Mac:           Command+Shift+P (Play/Pause)
                Command+Shift+S (Stop)

Troubleshooting

Tidak ada suara:
   - Pastikan volume komputer tidak mute
- Cek setting suara di sistem operasi
- Restart Chrome

Teks tidak terbaca:
- Refresh halaman
- Pastikan teks bisa diblok
- Coba di website lain

Shortcut tidak berfungsi:
- Buka chrome://extensions/shortcuts
- Reset shortcut ke default
- Cek konflik dengan extension lain
